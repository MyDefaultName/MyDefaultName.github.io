Part 4 - Java For Loops

In Java, we use for loops when we want the program to repeat something a certain number of times. It's like saying: “Do this again and again until we reach the end.”

Syntax:

for (initialization; condition; update) {
    // code runs again and again while the condition is true
}

Example:

public class NumberPrinter {
    public static void main(String[] args) {
        // This loop prints numbers from 1 to 5
        for (int i = 1; i <= 5; i++) {
            System.out.println("Number: " + i);
        }
    }
}

Explanation:

  • int i = 1; → We start counting from 1.
  • i <= 5; → We keep going while i is less than or equal to 5.
  • i++; → We add 1 to i every time the loop runs.
  • Inside the loop, we print the value of i.

Output:

Number: 1
Number: 2
Number: 3
Number: 4
Number: 5

In Simple Words:

A for loop helps us repeat code easily. Instead of writing System.out.println() many times, the loop does it for us.
