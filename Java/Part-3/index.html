Part 3 - Java If Else Statements

In Java, we use if else statements when we want the program to decide something based on a condition. It's like asking a question: "Is this true?" If yes, then do something. If not, do something else.

Syntax:

if (condition) {
    // code runs if condition is true
} else {
    // code runs if condition is false
}

Example:

public class GradeChecker {
    public static void main(String[] args) {
        int grade = 85;

        if (grade >= 75) {
            System.out.println("You passed");
        } else {
            System.out.println("You failed try again");
        }
    }
}

Explanation:

  • if (grade >= 75) - The program checks if the grade is 75 or higher.
  • If true, it shows "You passed!"
  • If false, it shows "You failed. Try again!"
