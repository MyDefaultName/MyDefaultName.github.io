Part 6 - Java Do-While Loop

A do-while loop lets us run a block of code first, and then check the condition to see if we should repeat. It’s like saying: “Do this at least once, then check if we need to do it again.”

Syntax:

do {
    // code runs at least once
} while (condition);

Example:

public class EvenNumberPrinter {
    public static void main(String[] args) {
        int number = 2;

        // This loop prints even numbers from 2 to 10  
        do {
            System.out.println("Even number: " + number);
            number += 2;
        } while (number <= 10);
    }
}

Explanation:

  • int number = 2 → Start at 2.
  • do { ... } → Run the block first.
  • System.out.println(...) → Show the current number.
  • number += 2 → Increase by 2 each time to get even numbers.
  • while (number <= 10) → Keep going while number is 10 or less.

Output:

Even number: 2
Even number: 4
Even number: 6
Even number: 8
Even number: 10

In Simple Words:

This do-while loop prints even numbers. It's useful when you want your code to run once before checking if it should continue. This saves time and keeps your code clean.
