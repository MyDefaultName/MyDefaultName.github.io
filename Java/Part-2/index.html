Part 2 - Variables and Data Types

In Java, a variable is like a container that stores data. Every variable has a type — which tells Java what kind of data the variable holds. For example, is it a number? A word? A true/false value?

Variable Syntax:

dataType variableName = value;

Common Data Types:

  • int: Whole numbers (e.g., 7, -42)
  • double: Decimal numbers (e.g., 3.14, -2.5)
  • char: Single character (e.g., 'A')
  • String: Text (e.g., "Hello!")
  • boolean: True or false values

Example:

public class InfoDemo {
    public static void main(String[] args) {
        int age = 20;
        double price = 99.99;
        char grade = 'A';
        String name = "Alexa";
        boolean isStudent = true;

        System.out.println(name + " is " + age + " years old.");
    }
}

Explanation:

  • int age = 20; defines an integer variable named age.
  • String name = "Alexa"; stores a name in text form.
  • System.out.println(...); prints the combined message to the console.
