Part 1 - Introduction to Java

Java is a statically-typed language, meaning you must declare the type of a variable before using it. The basic structure of a Java program involves classes and methods. A main method is the entry point for execution.

Example:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
  • public class HelloWorld: This declares a class named HelloWorld. The public keyword means it's accessible from anywhere. Class names typically start with a capital letter.
  • public static void main(String[] args): This is the main method. public makes it accessible from anywhere. static means it belongs to the class itself, not a specific object. void indicates it doesn't return a value. String[] args allows command-line arguments to be passed to the program.
  • System.out.println("Hello, World!"); This prints "Hello, World!" to the console. System.out is a static member of the System class that represents the standard output stream. println() is a method that prints a line of text.

Key Syntax Points:

  • Semicolons: Every statement in Java ends with a semicolon (;).
  • Braces: Curly braces {} define blocks of code, such as the body of a method or a loop.
  • Comments: // for single-line comments and /* ... */ for multi-line comments.
  • Case Sensitivity: Java is case-sensitive (HelloWorld is different from helloworld).
