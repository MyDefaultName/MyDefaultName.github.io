Part 5 - Java While Loops

A while loop checks the condition before running the block of code. If the condition is true, the loop runs. It keeps repeating until the condition becomes false.

Syntax:

while (condition) {
    // code runs if the condition is true
}

Example:

public class OddNumberPrinter {
    public static void main(String[] args) {
        int number = 1;

        // This loop prints odd numbers from 1 to 9  
        while (number < 10) {
            System.out.println("Odd number: " + number);
            number += 2;
        }
    }
}

Explanation:

  • int number = 1 → Start at 1 (first odd number).
  • while (number < 10) → Run the loop while the number is less than 10.
  • System.out.println(...) → Print the current odd number.
  • number += 2 → Increase by 2 to get the next odd number.

Output:

Odd number: 1
Odd number: 3
Odd number: 5
Odd number: 7
Odd number: 9

In Simple Words:

This program uses a while loop to print odd numbers from 1 to 9. Since odd numbers aren't divisible by 2, we start at 1 and increase by 2 each time. The loop stops when the number reaches 10 or more.
